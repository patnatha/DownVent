#Ventilator Logger
This is the home directory for the serial COM/USB based logger for the Datex-Ohmeda Aestiva/5 7900 ventilator. The protocol is based on the Ohmeda COM 1.0 Serial Protocol, Version 1.5 (August 14th, 2001). 

In the home directory there must be a token.auth file with the redcap connection details which include the first line is the "OR" string. Second line is the renpoint (HTTPS://...), and the third line is the token which must be kept a secret.
